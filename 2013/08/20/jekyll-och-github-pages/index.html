Jekyll och Github Pages

Aug 20th, 2013

Denna blogg har fått en ny form och även flyttat till en ny plattform. Kanske kan detta innebära att frekvensen av uppdateringar ökar något?

Bloggen görs nu tillgänglig via Jekyll, som skiljer sig markant från tidigare använda WordPress. Ingen databas används, istället genereras statiska HTML-filer som serveras till dig från Github Pages. Inläggen har jag valt att skriva med Markdown.

Nästa steg är att importera alla mina tidigare blogginlägg, daterade tillbaka till 2009, från WordPress. Största anledningen till flytten är att jag helt enkelt ville pröva något nytt.
